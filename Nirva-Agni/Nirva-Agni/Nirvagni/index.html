About Nirva - Agni

NIRVA-AGNI is a device which is a two-axis robotic arm placed on top to the roof uses computer vision(camera) and deep learning algorithm to detect and extinguish fires in real time. It automatically triggers the water with less precision. The device also sends real-time alerts like message and a photo to occupants and authorities. It aims to improve fire detection response times, minimize damage, efficiency and ensure safety with minimal human intervention. It requires less maintenance.
